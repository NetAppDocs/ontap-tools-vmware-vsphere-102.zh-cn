= 更改ONTAP工具管理器密码
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用ONTAP工具管理器更改管理员密码。

* 步骤 *

. 从Web浏览器启动ONTAP工具管理器： `\https://loadBalanceIP:8443/virtualization/ui/`
. 使用您在部署期间提供的适用于VMware vSphere的ONTAP工具管理员凭据登录。
. 单击屏幕右上角的*管理员*图标，然后选择*更改密码*。
. 在更改密码弹出窗口中、输入旧密码和新密码详细信息。更改密码的限制将显示在UI屏幕上。
. 单击*Change*以实施更改。
